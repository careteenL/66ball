## ES6的骚操作

### 目录

- var function
  - var存在的问题 
    - 会变量提升（先声明，不会复制）
    - 没有块级作用域，是全局作用域/函数作用域
- let const  
  - let的优势 
    - 不会变量提升，不能重复被定义，不会污染全局变量
    - 会和{}产生作用域
    - 
  - let存在一定问题
```js
let a = 2
{
  console.log(a)
  let a =1
}
// undefined
  ```  
  - const
    - let 可以重新赋值 const不能改变赋值的空间
```js
const a = 1
a = 2
// 报错
const b = []
b.push(1)
// 正常运行
```
